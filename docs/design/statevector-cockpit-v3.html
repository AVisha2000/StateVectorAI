What should I run next on the contextual task?
Your strongest open thread is sample efficiency on contextual. The QRNN candidate holds at 4q, but nothing tests it near the barren-plateau onset. Larocca 2025 argues reuploading ansätze resist plateaus longer than hardware-efficient — so the decisive test is reuploading at 8–12 qubits.
Any reason it wouldn't replicate?
Two risks. It's a single task instance, so the edge could be dataset-specific — pair it with the interference dataset for a second instance. And Cerezo 2021 notes cost-landscape effects can masquerade as sample efficiency; the frozen-circuit control already guards that.
